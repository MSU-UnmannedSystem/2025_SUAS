# /legacy_flight_architecture/ : Historical Archive

This directory serves as the long-term storage and historical record for the 2026 C-UASC campaign. It contains code that was superseded by the final forensics and scoring suite, or was used exclusively for offline preparation and hardware validation.

**Contents:**
* `/listener/`: Parallel telemetry logging tools used during flight testing.
* `/test/`: Placeholder and structural testing scripts.
* `/train/`: Offline training pipelines and dataset configurations for vision models.
